# 🧸 Sistema de Cadastro - Cordeirinhos

![Status](https://img.shields.io/badge/Status-Em_Produção-success)
![Python](https://img.shields.io/badge/Python-3.10-blue)
![Django](https://img.shields.io/badge/Django-5.2-green)

Um sistema web desenvolvido para automatizar e organizar o cadastro de crianças e responsáveis do Departamento Infantil Cordeirinhos. O projeto foca em uma interface amigável, armazenamento seguro de dados e facilidade de uso para a equipe do departamento.

## 🚀 Funcionalidades

* **Cadastro Completo:** Registro de dados da criança, incluindo fotos 3x4 e datas de nascimento.
* **Contatos dos Responsáveis:** Armazenamento estruturado de telefones principais e alternativos (WhatsApp).
* **Interface Responsiva:** Design limpo e temático, com validação de formulários.
* **Deploy em Nuvem:** Sistema hospedado e rodando em ambiente de produção (PythonAnywhere).

## 🛠️ Tecnologias Utilizadas

* **Back-end:** Python, Django
* **Front-end:** HTML5, CSS3
* **Banco de Dados:** SQLite (Desenvolvimento/Produção inicial)
* **Hospedagem:** PythonAnywhere
